Free Online Line Number Tool

Add line numbers to any text, code, or document instantly. This tool prepends sequential numbers to each line with customizable formatting — choose your starting number, step increment, separator style, and whether to pad with leading zeros. Useful for code review, document referencing, legal text, and data preparation.

Formatting Options

Start at any number (default is 1, but you can start at 0 or any custom value). Set a step increment (every 1, every 5, every 10, etc.). Choose from 6 separator styles: dot, colon, parenthesis, pipe, tab, or double space. Enable zero-padding for aligned columns (001, 002... instead of 1, 2...). Skip empty lines to keep your numbering tight.

Use Cases

Code review: add line numbers to code snippets for discussion ("see line 42"). Legal documents: reference specific lines in contracts and agreements. Data processing: number rows before importing into spreadsheets. Teaching: create numbered exercise lists or quiz questions. Troubleshooting: add line numbers to log files for easier debugging.

Frequently Asked Questions

Can I start numbering from a specific number?

Yes. Change the "Start at" value to any number. You can also set a custom step — for example, starting at 10 with a step of 10 gives you 10, 20, 30, etc.

What does zero-padding do?

Zero-padding adds leading zeros so all line numbers have the same width. For example, in a 100-line text, lines are numbered 001, 002... 099, 100 instead of 1, 2... 99, 100. This keeps the text neatly aligned.
