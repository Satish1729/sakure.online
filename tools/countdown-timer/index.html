Frequently Asked Questions

Does the countdown keep running if I close the tab? ▼
The countdown is based on a saved target date, not an active timer. When you return to the page, it recalculates automatically — the countdown will always show the correct remaining time regardless of how long the tab was closed.
How many countdowns can I save? ▼
You can save up to 8 countdowns. They are stored in your browser's localStorage and will persist across page reloads. Clearing browser data will remove them.
What happens when a countdown reaches zero? ▼
The display shows a "🎉 Time's Up!" message. Past events show negative elapsed time so you can see how long ago they occurred.
Does this work without an internet connection? ▼
Once the page is loaded, the countdown runs entirely in your browser using JavaScript — no internet connection required. However, refreshing the page requires the fonts to load again if you're offline.
