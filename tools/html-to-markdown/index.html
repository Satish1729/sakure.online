Frequently Asked Questions

What HTML elements are supported?

All common elements: h1–h6, p, br, strong/b, em/i, del/s, a, img, pre/code, ul/ol/li, blockquote, table/thead/tbody/tr/th/td, hr, and div/span containers. Unknown elements fall back to their inner text.

What happens to inline styles or CSS classes?

Inline styles and CSS classes are stripped — Markdown doesn't support them natively. The tool preserves semantic meaning (bold, italic, links) but discards visual-only attributes.

Can I convert full HTML pages?

Yes. Paste a full page including <html>, <head>, and <body> tags. The converter extracts the body content automatically and ignores the head section.

How are nested lists handled?

Nested lists (ul/ol inside li) are indented with 2 spaces per level, which is standard Markdown. Both ordered and unordered nesting work correctly.

Is any data sent to a server?

No. All conversion happens entirely in your browser using JavaScript. Nothing leaves your machine, so it's safe to use with sensitive or proprietary HTML content.
