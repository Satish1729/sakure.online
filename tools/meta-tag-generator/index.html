Frequently Asked Questions

What are meta tags and why do they matter?

Meta tags are HTML snippets placed in your page's <head> that tell search engines and social media platforms what your page is about. Good meta tags improve click-through rates from search results and control how your link looks when shared on platforms like Facebook or Twitter.

What is the ideal length for a meta title and description?

Keep your title under 60 characters (ideal: 50–60) and your description between 150–160 characters. This tool shows a live character counter with colour-coded warnings to help you stay within the optimal range.

What is an Open Graph image and what size should it be?

The Open Graph image is the thumbnail shown when your link is shared on social media. The recommended size is 1200×630 pixels (1.91:1 ratio). For Twitter's summary_large_image card, the same dimensions work perfectly.

Do I need a robots meta tag if I want my page indexed?

No. The default behaviour for search engines is "index, follow", so you only need a robots meta tag if you want to opt out of indexing or following links. Adding "index, follow" explicitly is redundant but not harmful.

Is the canonical tag the same as the URL meta tag?

No. The canonical tag is a <link rel="canonical"> element — not a <meta> tag — that tells search engines which URL is the primary version of a page when duplicate content exists. This tool generates the canonical link element correctly.
